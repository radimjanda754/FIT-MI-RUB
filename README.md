# MI-RUB Homework repository

## Requirements

   * ruby 2.0 or higer

     ruby -v
     
   * `rspec` gem

     gem install rspec

   * `rubocop` gem

    gem install rubocop

   * run all tests from the `spec` directory

    rspec spec

   * assess quality of all your code

    rubocop *.rb
